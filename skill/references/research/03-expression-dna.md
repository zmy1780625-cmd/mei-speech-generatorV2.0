## 三、论述DNA

### 3.1 政策先行模式
在正式场合中（大会/研讨会），石梅会先引用政策背景再展开自身观点：

> "教育部怀进鹏部长强调：'加快教育大模型建设与"人工智能+教育"场景创新……'"
> → "腾讯作为长期深耕科技与教育领域的一员，积极响应国家政策……"

**结构**：国家政策方向 → 腾讯积极响应 → 具体行动

### 3.2 趋势判断模式
- "当前，我们正站在一场由人工智能驱动的教育变革浪潮之巅"
- "AI不再是遥远的概念，而是正在重构教与学的生态"
- "AI渗透了教学、管理的各个场景，实现了从技术探索到规模化应用的跨越式发展"
- "从覆盖广泛场景的通用大模型，到聚焦专业领域的学科大模型"

**DNA**：从宏观趋势切入 → 用对比/递进强化 → 落到教育场景

### 3.3 成果展示模式（密集数据支撑）
石梅在描述成果时，习惯用**时间线+数据**的方式逐条呈现：

> "25年8月，腾讯青少年AIGC创作工坊……短短两个月就覆盖全国**300多所学校**，服务**5000余名师生**，累计产出**3万余件**AIGC作品。"
> "25年9月，我们正式发布了腾讯教育智能体平台……提供**上百个**专家智能体和工具智能体……"
> "吸引来自**8所高校**的**64支战队**、**109名选手**参与，产出**100+**教育智能体应用成果。"

### 3.4 产品描述模式
描述新产品/平台时的固定结构：

```
产品名称 → 核心定位 → 功能模块 → 场景价值
```

示例：
> "教育智能体平台提供专家智能体、工具智能体、知识库、课程资源和赛事**五大核心模块**，汇聚上百个专家智能体和工具智能体，并且支持多智能体协作与资源调度。"

### 3.5 合作描述模式
描述校企合作时，强调双方互补：

> "腾讯云将提供强大的云计算基础设施和高性能的模型训练框架，而同济大学则以其深厚的教育理论积淀和丰富的教学实践场景，主导模型的教育内涵设计与应用范式创新。"

**DNA**：腾讯出技术底座 + 高校出教育内涵 → 双方共创
